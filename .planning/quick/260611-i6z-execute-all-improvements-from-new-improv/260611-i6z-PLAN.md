**5b. Implement in each adapter:**

**MySQL/MariaDB** (`internal/db/mysql/adapter.go`, `internal/db/mariadb/adapter.go`):
```go
func (a *MySQLAdapter) TestConnect(ctx context.Context, dsn string) error {
    db, err := sql.Open("mysql", dsn)
    if err != nil {
        return err
    }
    defer db.Close()
    ctx, cancel := context.WithTimeout(ctx, 3*time.Second)
    defer cancel()
    return db.PingContext(ctx)
}
```
(MariaDB uses the same `go-sql-driver/mysql` driver, so same implementation.)

**PostgreSQL** (`internal/db/postgresql/adapter.go`):
```go
func (a *PostgreSQLAdapter) TestConnect(ctx context.Context, dsn string) error {

    if err != nil {
        return err
    }

    return nil
}
```
