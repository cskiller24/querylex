STEP 10: Construct SQL (MySQL dialect)
---
Key decisions:
  - Tables: employees, salaries, dept_emp, departments
  - Join: employees → salaries (emp_no, to_date='9999-01-01' for current salary)
  - Join: employees → dept_emp (emp_no, to_date='9999-01-01' for current dept)
  - Join: dept_emp → departments (dept_no)
  - Filter: hire_date >= DATE_SUB(CURDATE(), INTERVAL 5 YEAR) — sargable
  - Order: hire_date DESC

SQL:
SELECT e.emp_no, e.first_name, e.last_name, e.hire_date,
       s.salary AS current_salary,
       d.dept_name AS department
FROM employees e
INNER JOIN salaries s ON e.emp_no = s.emp_no
    AND s.to_date = '9999-01-01'
INNER JOIN dept_emp de ON e.emp_no = de.emp_no
    AND de.to_date = '9999-01-01'
INNER JOIN departments d ON de.dept_no = d.dept_no
WHERE e.hire_date >= DATE_SUB(CURDATE(), INTERVAL 5 YEAR)
ORDER BY e.hire_date DESC
