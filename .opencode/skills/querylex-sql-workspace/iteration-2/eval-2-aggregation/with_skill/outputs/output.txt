```sql
SELECT d.dept_no, d.dept_name, AVG(s.salary) AS avg_salary
FROM departments d
INNER JOIN dept_emp de ON d.dept_no = de.dept_no AND de.to_date = '9999-01-01'
INNER JOIN salaries s ON de.emp_no = s.emp_no AND s.to_date = '9999-01-01'
GROUP BY d.dept_no, d.dept_name
ORDER BY avg_salary DESC;
```

Joins departments → dept_emp (on dept_no, confidence 1.0 FK) then dept_emp → salaries (on emp_no, via the join graph path). Both temporal tables (dept_emp, salaries) are filtered to current rows only (to_date = '9999-01-01'). Groups by department and computes average salary.
