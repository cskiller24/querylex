```sql
SELECT d.dept_name, AVG(s.salary) AS avg_salary
FROM departments d
INNER JOIN dept_emp de ON d.dept_no = de.dept_no AND de.to_date = '9999-01-01'
INNER JOIN salaries s ON de.emp_no = s.emp_no AND s.to_date = '9999-01-01'
GROUP BY d.dept_name
ORDER BY avg_salary DESC
```

**Explanation:** Joins `departments` → `dept_emp` → `salaries` via foreign keys (dept_no, emp_no). Filters both `dept_emp` and `salaries` on `to_date = '9999-01-01'` to get only current department assignments and current salaries (standard employees database temporal pattern). Groups by `dept_name` with `AVG(s.salary)` aggregation, ordered descending.
