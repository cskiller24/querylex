```sql
SELECT
  d.dept_no,
  d.dept_name,
  ranked.emp_no,
  ranked.first_name,
  ranked.last_name,
  ranked.salary
FROM (
  SELECT
    e.emp_no,
    e.first_name,
    e.last_name,
    de.dept_no,
    s.salary,
    ROW_NUMBER() OVER (PARTITION BY de.dept_no ORDER BY s.salary DESC) AS rn
  FROM employees e
  INNER JOIN dept_emp de ON e.emp_no = de.emp_no
    AND de.to_date = '9999-01-01'
  INNER JOIN salaries s ON e.emp_no = s.emp_no
    AND s.to_date = '9999-01-01'
) ranked
INNER JOIN departments d ON ranked.dept_no = d.dept_no
WHERE ranked.rn = 1
ORDER BY d.dept_no;
```

Uses MySQL `ROW_NUMBER()` window function, partitioned by department and ordered by salary descending. Filters `dept_emp.to_date = '9999-01-01'` and `salaries.to_date = '9999-01-01'` for current assignments only. Joins employees → dept_emp → departments and employees → salaries via foreign-key-backed paths (confidence=1).
